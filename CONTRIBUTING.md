# Contributing

Thanks for your interest in improving Letterbox Lab!

## How to Contribute
1. Fork the repo
2. Create a new branch
3. Make your changes
4. Open a Pull Request

## Development
This is a pure front-end project. No build step is required. Just open `index.html` in a browser.

## Code Style
- Keep the UI lightweight
- Prefer small, focused changes
- Avoid adding heavy dependencies

## Reporting Issues
Please include:
- Browser version
- Steps to reproduce
- Expected vs. actual behavior
- Screenshots if helpful
