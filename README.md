# Letterbox Lab (Add Letterbox to Photo)

一个在浏览器中完成“裁切 + 遮幅 + 水印”的轻量工具。无需安装，打开 `index.html` 即可使用。

## Features
- 拖拽/选择图片导入
- 裁切工具（可关闭裁切）
- 多种遮幅比例与自定义比例
- 图片上下偏移
- PNG 水印（位置/缩放/不透明度控制）
- 一键导出 PNG

## Quick Start
1. 直接双击打开 `index.html`
2. 导入图片
3. 选择裁切比例（可关闭裁切）
4. 选择遮幅比例与样式
5. 可选添加水印
6. 点击“下载 PNG”

## Project Structure
- `index.html` UI 结构
- `styles.css` 页面样式
- `app.js` 核心逻辑（绘制、裁切、遮幅、水印、导出）

## Notes
- 当前为纯前端项目，无需构建与依赖安装
- 建议使用现代浏览器（Chrome / Edge / Safari 最新版）
